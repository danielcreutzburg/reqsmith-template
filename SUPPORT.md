# Support

Thanks for using the ReqSmith template.

## Questions and general help

- Open a GitHub issue in this repository.
- Use the `question` label for usage/setup questions.

## Bug reports

- Please use the bug report template under `.github/ISSUE_TEMPLATE/`.
- Include reproduction steps, expected behavior, and environment details.

## Security issues

- Do **not** open public issues for vulnerabilities.
- Follow the process in `docs/SECURITY.md`.
